        Instantiate a new worker for given task data
        Parameters:
        classifier - the classifier indicating the type of message it is
        version - the api version of the task's message
        status - the status of the task
        data - the raw serialised task data
        context - provides access to task specific context, may be null
        tracking - additional fields used in tracking task messages
        Returns:
        a new Worker instance that will perform work upon the taskData
        Throws:
        TaskRejectedException - if a Worker cannot be created to handle this task currently
        InvalidTaskException - if it appears this task cannot possibly be handled by a Worker of this type
