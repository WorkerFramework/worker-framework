com.hpe.caf.worker.core

Class WorkerApplication

  • java.lang.Object
    • io.dropwizard.Application<WorkerConfiguration>
      • com.hpe.caf.worker.core.WorkerApplication

  • public final class WorkerApplication
extends io.dropwizard.Application<WorkerConfiguration>
    This is the main HP SaaS asynchronous micro-service worker entry point. On startup, it will identify implementations of necessary components (such as a queue, data store, and backend worker), and create a WorkerCore object which will handle the logic of data flow between these components. The application class itself here just wrappers this and exposes health checks and metrics.
